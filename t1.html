Process of Making Tea

  1. Take a pan and pour one cup of water into it.
  2. Place the pan on the stove and boil the water.
  3. Add one teaspoon of tea powder to the boiling water.
  4. Let it boil for 2-3 minutes until the color changes.
  5. Add milk according to your taste.
  6. Add sugar as required.
  7. Boil the tea for another 2 minutes.
  8. Turn off the stove.
  9. Strain the tea into a cup.
  10. The tea is now ready to drink.


Programming Language

HTML:
HyperText Markup Language,HTML is used to create web pages.It is used to structure the content of a website. HTML helps in adding text, images, videos, and links to a web page. It is used to create headings, paragraphs, lists, and tables.
CSS:
CSS stands for Cascading Style Sheets,CSS is used to style and design web pages. It controls how HTML elements look on the screen.It is used to add spacing, borders, and backgrounds
Java Script:
JavaScript is used to add interactivity to websites. It is used to create dynamic content on web pages. JavaScript is used to validate forms. It helps in creating animations and effects
